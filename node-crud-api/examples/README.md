# Примеры использования Memory Bank

В этой директории содержатся примеры использования Memory Bank в различных сценариях.

## Доступные примеры

### 1. memory-bank-example.js

Базовый пример, демонстрирующий все возможности Memory Bank:

- Создание экземпляра с настройками
- Сохранение и получение данных
- Работа с TTL (временем жизни)
- Автоматическая очистка и вытеснение
- Статистика использования
- События и мониторинг
- Работа с коллекциями

**Запуск:**
```bash
node node-crud-api/examples/memory-bank-example.js
```
